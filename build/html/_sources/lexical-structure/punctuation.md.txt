# 1.9. Punctuation
The following characters belong to the *punctuation character* group:
- {uc}`0028| ( ` {scmp}`left round bracket`
- {uc}`0029| ) ` {scmp}`right round bracket`
- {uc}`005b| [ ` {scmp}`left square bracket`
- {uc}`005d| ] ` {scmp}`right square bracket`
- {uc}`007b| { ` {scmp}`left curly bracket`
- {uc}`007d| } ` {scmp}`right curly bracket`
- {uc}`003b| ; ` {scmp}`semicolon`

These characters cannot be used to define operators and cannot be part of tokens other than character and string literals. Each punctuation character equals a token.

Opening (left) brackets must always be followed by closing (right) brackets of the same type.
