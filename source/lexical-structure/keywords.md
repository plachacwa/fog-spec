# 1.2. Keywords

*Keywords* are the special words that can be used only in limited contexts. They cannot be used as names for any entities inside or outside of the program.

```{note}
If you need to name or to reference an entity homonymous to a keyword, you can do it with the built-in symbol type, e.g., `sym "name of keyword"`.
```
